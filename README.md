## Run Locally

Clone the project

```bash
  git clone https://github.com/Deadreyo/SemiColon-registration-frontend.git
```

Go to the project directory

```bash
  cd SemiColon-registration-frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
